"What should I expect?"

It's a self-directed learning environment where you bring your own laptop to learn. Work on nodeschool.io tutorials or on a personal project you are working on! Mentors help you when you hit roadblocks.

"What is a tutorial? What's a workshopper?"

Lesson module, called workshopper; it is a tutorial, self-guided. Run it at home. Run it on a laptop. Run it offline too. There are eight core tutorials = workshoppers and 50 more. Created by the Nodeschool community. They're opensourced, obviously :-)

"How do I do a tutorial?"

To set up and run the Beginner Node Workshopper, open a terminal and type:

$ npm install -g learnyounode
$ learnyounode

Having trouble installing Node? No worries — we'll help you get set up and get it running once you arrive. The goal of NodeSchool is to help people explore and learn Node.
